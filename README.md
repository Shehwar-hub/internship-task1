# Rock, Paper, Scissors Game

A simple Python console-based Rock, Paper, Scissors game where you play against the computer. The game continues until you type `end` to quit.

## Features

1. Play against the computer
2. Random computer moves using the `random` module
3. Continuous gameplay with a `while` loop
4. Determines the winner automatically
5. Type `end` to exit the game
6. Handles invalid inputs

## Built With

- Python 3
- `random` module
